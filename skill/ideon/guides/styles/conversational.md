# Style: Conversational

## Best Fit
Threads, lightweight explainers, Q-and-A style posts.

## Rules

### Rule: Write Like You Speak, Edit Like You Publish
Description: Keep natural flow, then tighten for precision.
Negative example: Unedited speech fragments with unclear point.
Positive example: Natural sentence rhythm with clear claims and examples.

### Rule: Ask Useful Questions
Description: Questions should move the reader forward.
Negative example: "Isn't that cool?"
Positive example: "What changes if your opener states the outcome in one line?"

### Rule: Keep Informal Without Becoming Loose
Description: Casual tone should not reduce structural quality.
Negative example: Wandering narrative with no sections.
Positive example: Short sections with clear transitions and one main point each.
