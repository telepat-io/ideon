# References Directory

This directory contains reusable concept docs referenced by multiple format and style guides.

## Files

- `prose-quality-checks.md`: clarity, consistency, and quality rule tiers
- `readability-and-pace.md`: readability targets and pacing guidance
- `x-thread-hooks.md`: lead-in hook framework for X threads
- `headline-writing-systems.md`: headline clarity, specificity, and variation workflow
- `skimmability-patterns.md`: structure and scan-first formatting patterns
- `ideation-and-credibility-systems.md`: idea generation and credibility framing
- `content-frameworks.md`: 3-bucket planning, 4A ideation, and curiosity-gap constraints
- `truthful-value-framing.md`: ethical persuasion patterns for reader-value framing
- `emotional-resonance.md`: emotion-first framing patterns for shareable but useful writing
- `bibliography-and-influence-map.md`: source synthesis and thematic mapping
- `target-length-guidance.md`: size tiers by channel and proportional structure rules
- `multi-channel-brief-strategy.md`: shared-brief model for primary and secondary outputs

## Usage Rule

If a concept appears in three or more guides, document it here and link to it from format/style pages rather than duplicating long explanations.
