# Styles Directory

This directory defines how the writing sounds.

Use this after choosing a format and content-intent.

- Format = where it publishes and how it is packaged
- Content-intent = what outcome the piece drives
- Style = how the writing sounds

## How To Use

1. Pick your target medium in `formats/`.
2. Pick one primary intent in `content-intent/`.
3. Pick one primary style here.
4. Apply shared quality checks in `references/`.
