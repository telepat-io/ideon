# Style Selection Matrix

Use this matrix to choose a default style quickly.

| Goal | Recommended Primary Style | Common Secondary Blend |
|---|---|---|
| Executive clarity | professional | authoritative |
| Teach complex process | technical | empathetic |
| Formal evidence-led analysis | academic | analytical |
| Convince on strategy change | persuasive | professional |
| Build trust in community updates | friendly | conversational |
| Fast factual summary | journalistic | minimalist |
| Humanized onboarding | empathetic | friendly |
| Sharp short-form guidance | minimalist | authoritative |

## Rule: Pick One Primary Style
Description: Start with one base style, then add one supporting style only if needed.
Negative example: Attempting four styles in one short post.
Positive example: Primary technical style with empathetic support lines.
