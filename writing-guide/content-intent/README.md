# Content-Intent Directory

This directory defines what a piece is trying to accomplish.

Use this after choosing a format and before locking style.

- Format = where it publishes and how it is packaged
- Content-intent = what outcome the piece drives
- Style = how the writing sounds

## How To Use

1. Pick your target medium in `formats/`.
2. Pick one primary intent in this directory.
3. Pick a style in `styles/`.
4. Apply shared quality checks in `references/`.
