# Formats Directory

This directory defines where a piece publishes and how it is packaged.

Use this as your first pick before choosing content-intent and style.

- Format = where it publishes and how it is packaged
- Content-intent = what outcome the piece drives
- Style = how the writing sounds

## Format Playbooks

- [article.md](article.md)
- [blog-post.md](blog-post.md)
- [newsletter.md](newsletter.md)
- [x-post.md](x-post.md)
- [x-thread.md](x-thread.md)
- [reddit-post.md](reddit-post.md)
- [linkedin-post.md](linkedin-post.md)
- [press-release.md](press-release.md)
- [science-paper.md](science-paper.md)

## How To Use

1. Pick your target medium here.
2. Pick one primary intent in `content-intent/`.
3. Pick a style in `styles/`.
4. Apply shared quality checks in `references/`.
